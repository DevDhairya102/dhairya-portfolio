## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/dhairya-portfolio.git
```

### Navigate to Project

```bash
cd dhairya-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
